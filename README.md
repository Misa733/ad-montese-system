# AD Montese System

Sistema administrativo ministerial da Assembleia de Deus Montese.

## Tecnologias

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Google Sheets API

## Comandos

```bash
npm install
npm run dev
npm run build
```

## Configuracao

Copie `.env.example` para `.env` e configure as credenciais da Google Sheets API.
